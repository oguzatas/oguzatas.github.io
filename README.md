## Customization

### Colors
The website uses CSS custom properties (variables) that you can easily customize in `styles.css`:

```css
:root {
    --primary-color: #8B4513;    /* Main brown color */
    --secondary-color: #D4AF37;  /* Gold accent */
    --accent-color: #B8860B;     /* Dark gold */
    --dark-color: #2C1810;       /* Dark brown */
    --light-color: #F5F5DC;      /* Light background */
}
```
